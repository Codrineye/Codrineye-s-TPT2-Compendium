# winAI

winAI is a blueprint AI made by WootImNoob for the blueprint of the same name.<br>
It was originally 8 scripts long, but now that it uses the stand alone compactor, it's only 4 scripts big.

## Source import

winAI was not made as a source program, but I've reworked it so that anyone can look at it for a practical example.
