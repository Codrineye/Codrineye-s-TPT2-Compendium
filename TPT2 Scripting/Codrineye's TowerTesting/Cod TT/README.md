# Cod TT

Cod TT is a rework of the original Codrineye's TowerTesting version 0.5 made by bluecat.<br>
I am still immensely grateful to bluecat for even making an attempt to make sense of that old script.

This folder is to keep `Cod TT` on the repo, in a separate folder as it deserves.<br>
